# CS Career Path Advisor

A web project that helps Computer Science students choose a suitable career path.

## Live Demo

https://mohamed-ashraf-ds.github.io/CS-Career-Path-Advisor/

## Project Description

CS Career Path Advisor is a website designed for Computer Science students who are confused about which specialization to choose.

The website shows different tech career paths, required skills, tools, job roles, roadmaps, suggested projects, and a global demand dashboard.

## Features

- Career Quiz to recommend a suitable specialization
- Career Explorer for different Computer Science fields
- Roadmap for each specialization
- Required skills and tools
- Suggested projects for practice
- Global Demand Dashboard
- Region filter: Global, Egypt, USA, Europe, Gulf
- Save favorite career path using localStorage
- Dark Mode
- Search bar

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages

## Note

The demand dashboard uses sample educational data stored in JavaScript to demonstrate dashboard logic.  
It is not based on real-time job market data.
